# Mintify Landing Page

This is a simple landing page project that I built while learning frontend development. The goal of this project was to practice HTML and CSS by recreating a modern-style landing page layout with different sections, cards, and media elements.

## What I Learned

While building this project, I learned several new concepts:

* How to use **mask-image** to create fade effects.
* How to control **opacity** for better visual design.
* How to use the **video tag** in HTML.
* How to design and structure **different types of cards**.
* How to organize sections of a landing page properly.
